Project Overview
This project demonstrates the creation of a small relational database using SQLite. The database consists of three tables: Authors, Books, and Quotes. The table structure designers created each table with field specifications and established relationships through foreign key connections. The project aims to develop skills in table design and structured data insertion and SQL query writing for retrieving related data.
The Authors table stores author details such as name, nationality, and birth year. The Books table stores book information including title, genre, published year, and a reference to the author. The Quotes table stores individual quotes and links them to both books and authors.
Implemented Features
•	Created three relational tables
•	Inserted 5 authors
•	Inserted 10 books
•	Inserted 20 quotes
•	Used foreign key relationships
•	Wrote SQL queries to:
o	Count quotes per book
o	Count quotes per author
o	Retrieve quotes by specific book
o	Retrieve quotes by specific author

The database design process enhanced my comprehension of backend data management systems and their corresponding query execution processes.
